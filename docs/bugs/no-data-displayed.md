# No data displayed

This bug has been seen on prod environment once.

Logged in to the app as a test user, concept data was being displayed as expected but at some point it stopped being displayed. the different views displayed the default messages saying that no concepts are available. this was the same for all views. cliccking on the navigation bar to see all the different views resulted in these default messages with no concept elements being displayed. refreshing the pages did not fix the issue. only after logging out and logging back in the data was displayed.

We need to see if there is an obvious root cause for this issue or if it may just have been a connection type of issue.
